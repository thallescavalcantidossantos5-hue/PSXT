# PSXT - PS2 Emulator with Beautiful UI

Projeto para criar um emulador de PS2 para Android com interface moderna e HUD inspirado no PPSSPP.

## Objetivo
- Fork de emulador PS2 (ARMSX2 ou NetherSX2)
- Trocar interface antiga por uma mais bonita e limpa
- Adaptar HUD/Touch controls do PPSSPP
- Manter core de emulação estável

## Divisão de Trabalho
- **Grok**: Estrutura, UI moderna, temas, menus
- **ChatGPT**: Adaptação do HUD do PPSSPP

Status: Iniciando projeto do zero.
