# PSXT - Instruções para IAs (Grok + ChatGPT)

## Objetivo
Criar um fork de emulador PS2 com interface e HUD bonito como o PPSSPP.

## Divisão
- Grok: Estrutura geral, UI moderna, temas, menus
- ChatGPT: Adaptação do HUD / Touch Controls do PPSSPP

## Regras
- Manter core de emulação intacto
- Priorizar simplicidade
- Trabalhar modularmente
