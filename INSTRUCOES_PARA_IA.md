# PSXT - Instruções para IAs (Grok + ChatGPT)

## Objetivo
Criar um fork de emulador PS2 para Android com interface e HUD bonitos, inspirados no PPSSPP.

## Divisão
- Grok: Estrutura geral do projeto, nova UI (temas, menus, navegação), simplificações
- ChatGPT: Adaptar o sistema de HUD / Touch Controls do PPSSPP

## Regras
- Manter o core de emulação o mais intacto possível
- Priorizar simplicidade e performance
- Remover animações desnecessárias
- Trabalhar modularmente
